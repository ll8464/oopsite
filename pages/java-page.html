Warm Up With Java!

Spring Boot

Spring is an opinionated Java framework that simplifies development. It allows full-stack applications to be setup quickier and easier by removing the need for boilerplate code and includes an embedded Tomcat server via Spring Boot. The numberous features of Spring allows the developer to focus on the business layer instead of configuration.

Spring utilizes annotations, such as @Component or @Repository, to interact with your code. By declaring the appropriate annotations, Spring can configure your code and inject dependencies as needed.

How to Get Started

To get started with Spring Boot, simply go to start.spring.io . This will take you to the spring initializr which lets you set your dependencies and project type to quick start your project. I recommend using Maven for the project type and at least Spring Boot DevTools and Spring Web as dependencies. Spring Boot DevTools allows for live reloads which makes testing much easier and Spring Web includes Tomcat server.

Spring Definitions

The following are helpful definitions to keep in mind when using Spring inspired by Frank Moley.

POJO - Plain old Java Object
Java Beans - Objects with only getters and setters
Spring Beans - POJOs configured in the application context. They are handled by Spring itself.
DTO - Data Transfer Object - Bean is used to move state between application layers.
