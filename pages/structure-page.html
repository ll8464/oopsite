Data Structures

Data structures are used for the storage and processing of data. There are multiple types, each with their advantages and disadvantages.

The syntax and some implementations vary depending on the programming language. It is best to review the documentation of the language of your choice when utilitizing data structures.

Common, must know data structures include arrays, stack, queue, linked lists and binary trees.
